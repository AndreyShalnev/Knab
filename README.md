## Coding Assignment

The assignment is the following:
Create an application that accepts a cryptocurrency code as input. The application should then display its current quote in the following currencies:

 - USD

 - EUR

 - BRL

 - GBP

 - AUD

To complete this task, you can only use these 2 APIs:
 
 - https://exchangeratesapi.io
 
 - https://coinmarketcap.com/api (free version)
